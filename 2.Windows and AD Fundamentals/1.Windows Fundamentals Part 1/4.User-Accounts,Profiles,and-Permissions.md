# User Accounts, Profiles, and Permissions

# User Account Types

## Administrator
- Can make system-wide changes
- Can:
  - Add/remove users
  - Modify groups
  - Change system settings
  - Install programs

## Standard User
- Limited permissions
- Can only:
  - Modify own files/folders
- Cannot:
  - Install software
  - Perform administrative tasks

---

# Viewing User Accounts

## Method 1: Other Users Settings

### Steps
1. Open **Start Menu**
2. Search:
   ```powershell
   Other User
   ```
3. Open:
   ```text
   Settings > Other users
   ```

## Options Available
- Add someone else to this PC
- Change account type
- Remove account

> Only Administrators can add new users.

---

# Changing Account Type

## Steps
1. Select local user
2. Click:
   ```text
   Change account type
   ```
3. Choose:
   - Administrator
   - Standard User

---

# User Profiles

## Profile Location
```powershell
C:\Users
```

Example:
```powershell
C:\Users\Max
```

## Profile Creation
- Created during first login
- Windows sets up:
  - Desktop
  - Documents
  - Downloads
  - Music
  - Pictures

## User Profile Service
- Windows service responsible for creating profiles during first login

---

# Common User Folders

# Local Users and Groups Management

## Open Tool

### Using Run Dialog
1. Press:
   ```powershell
   Win + R
   ```
2. Type:
   ```powershell
   lusrmgr.msc
   ```

## Main Sections
- Users
- Groups

---

# Groups

## Purpose
- Groups contain predefined permissions
- Users inherit permissions from assigned groups

## Important Points
- A user can belong to multiple groups
- Permissions depend on group membership

Example groups:
- Administrators
- Users
- Guests
