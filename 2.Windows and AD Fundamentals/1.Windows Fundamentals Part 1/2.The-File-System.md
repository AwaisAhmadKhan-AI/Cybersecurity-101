# Windows File System

## File Systems

### Common Windows File Systems
- FAT16 / FAT32
- HPFS
- NTFS (modern Windows default)

## FAT
- Common in:
  - USB drives
  - SD cards
- Limited compared to NTFS

---

# NTFS (New Technology File System)

## Features
- Journaling file system
- Can recover from failures using logs
- Supports files larger than 4GB
- File & folder permissions
- Compression
- Encryption (EFS)

---

# NTFS Permissions

## Permission Types
- Full Control
- Modify
- Read & Execute
- List Folder Contents
- Read
- Write

---

# Viewing Permissions

## Steps
1. Right-click file/folder
2. Select `Properties`
3. Open `Security` tab
4. Select user/group
5. View permissions

---

# Alternate Data Streams (ADS)

## ADS
- NTFS feature
- Allows hidden extra data inside files
- Every file has at least one stream (`$DATA`)

## Important Points
- Hidden in normal Windows Explorer
- Can be viewed with:
  - PowerShell
  - Third-party tools

## Security Relevance
- Malware can hide data in ADS
- Windows also uses ADS legitimately
  - Example: marking downloaded internet files
