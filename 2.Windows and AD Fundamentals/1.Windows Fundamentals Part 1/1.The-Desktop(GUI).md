# Windows Desktop (GUI)

## GUI
- GUI = Graphical User Interface
- Main screen after logging into Windows
- Login requires valid username & password

---

# Main Components

- Desktop
- Start Menu
- Search Box (Cortana)
- Task View
- Taskbar
- Toolbars
- Notification Area

---

# Desktop

- Stores shortcuts to:
  - Programs
  - Files
  - Folders
- Used for quick access

## Right-Click Options
- Arrange icons
- Change icon size
- Copy/Paste
- Create:
  - Folder
  - Shortcut
  - Text file

## Display Settings
- Change:
  - Resolution
  - Screen orientation
  - Multi-monitor setup

## Personalize
- Change:
  - Wallpaper
  - Fonts
  - Themes
  - Colors

---

# Start Menu

- Opened by clicking Windows logo
- Gives access to:
  - Apps
  - Files
  - Utilities
  - Settings

## Sections

### 1. Account & Power Options
- User settings
- Lock screen
- Sign out
- Documents
- Pictures
- Settings (gear icon)
- Power options:
  - Shutdown
  - Restart
  - Disconnect

### 2. Installed Applications
- Recently added apps
- All installed apps
- Alphabetically sorted

## Alphabet Grid
- Helps jump to app sections quickly

### 3. Tiles
- App shortcuts on right side
- Can:
  - Resize
  - Unpin
  - View properties

## Pin to Start
- Right-click app → "Pin to Start"

---

# Taskbar

- Shows running apps/programs
- Hovering shows:
  - Preview thumbnail
  - Tooltip

## Right-Click Taskbar
- Customize taskbar settings
- Enable/disable components

## Pinned Apps
- Stay visible even after closing

---

# Notification Area

- Bottom-right corner
- Shows:
  - Date & Time
  - Volume
  - Network/WiFi

## Customization
- Add/remove icons in Taskbar settings

---

# Important Tip

- Right-click almost anything for:
  - More options
  - Properties
  - Extra actions
