# User Account Control (UAC)

# What is UAC?
- Security feature introduced by Microsoft
- Helps prevent unauthorized system changes
- Protects systems from malware and accidental administrative actions

---

# Why UAC Exists

## Problem
- Many users run Windows as Administrators
- Malware running under an admin account can:
  - Install software
  - Modify system files
  - Change security settings

## Solution
- UAC limits elevated privileges unless explicitly approved

---

# How UAC Works

## Administrator Login
- Even Administrators run with standard privileges initially
- Elevated privileges are only granted when needed

## When Elevation is Required
Windows prompts the user before:
- Installing software
- Modifying system settings
- Running administrative tools

---

# UAC Prompt

## Trigger
- Appears when a task requires administrator privileges

## Standard User Behavior
- Must enter Administrator credentials

## Administrator Behavior
- Must confirm permission

---

# UAC Shield Icon

## Meaning
- Indicates the program requires elevated privileges

Example:
```text
Shield icon on installer/application
```

---

# Important Notes

## Built-in Administrator Account
- UAC is disabled by default for the built-in Administrator account

## Security Benefit
- Reduces chances of malware gaining full system control automatically

---

# Example Scenario

## Standard User Attempts Installation
1. User launches installer
2. UAC prompt appears
3. Requires Administrator password
4. Installation proceeds only after approval

If no password is entered:
- Prompt closes
- Installation fails

---

# Checking Permissions

## Steps
1. Right-click file/program
2. Select:
   ```text
   Properties
   ```
3. Open:
   ```text
   Security tab
   ```
4. View:
   - Users
   - Groups
   - Permissions

---

# Key Terms

| Term | Meaning |
|---|---|
| UAC | User Account Control |
| Elevated Privileges | Administrative permissions |
| Standard User | Limited access account |
| Administrator | Full system access |

---

# Security Advantages

- Prevents silent malware installation
- Limits unauthorized system changes
- Adds confirmation layer for sensitive operations
- Improves overall Windows security

---

| Admin Approval Mode | Limits default admin privileges |
