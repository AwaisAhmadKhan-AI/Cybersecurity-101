# Settings and Control Panel

# Overview

Windows uses two main interfaces for system configuration:

- Settings
- Control Panel

---

# Settings

## Introduction
- Introduced in Windows 8
- Designed for modern/touchscreen devices
- Main configuration interface in Windows 10+

## Common Uses
- Personalization
- Network settings
- Updates
- Privacy settings
- Display configuration

## Access
```text
Start Menu → Settings
```

---

# Control Panel

## Purpose
- Traditional Windows management interface
- Used for advanced system configurations

## Common Uses
- Uninstall programs
- Device management
- User account management
- Network adapter settings
- System administration tasks

## Access
```text
Start Menu → Control Panel
```

---

# Key Difference

| Settings | Control Panel |
|---|---|
| Modern interface | Traditional interface |
| Easier/basic settings | Advanced configurations |
| Touch-friendly | Administrative-focused |
| Default in Windows 10 | Legacy but still important |

---

# Installed Programs

## View Installed Applications

### Steps
1. Open:
   ```text
   Control Panel
   ```
2. Select:
   ```text
   Programs
   ```
3. Open:
   ```text
   Programs and Features
   ```

## Information Displayed
- Program name
- Publisher
- Version

---

# Example: Settings Redirecting to Control Panel

## Network Adapter Settings

### Steps
1. Open:
   ```text
   Settings
   ```
2. Go to:
   ```text
   Network & Internet
   ```
3. Click:
   ```text
   Change adapter options
   ```

### Result
- Opens a Control Panel window

> Some advanced settings still rely on Control Panel.

---

# Using Search

## Purpose
- Fastest way to find settings/tools

### Example
Search:
```text
wallpaper
```

Windows shows:
- Best matching settings
- Related control panel items
