# Task Manager

# Overview

Task Manager is a Windows utility used to monitor and manage:

- Running applications
- Background processes
- System performance
- Resource usage

---

# Main Uses

## Monitor System Resources
- CPU usage
- RAM usage
- Disk activity
- Network usage

## Manage Processes
- View running processes
- End unresponsive applications
- Identify high resource usage

## Performance Monitoring
- Real-time system statistics
- Hardware utilization tracking

---

# Opening Task Manager

## Method 1
- Right-click the taskbar
- Select:
  ```text
  Task Manager
  ```

## Shortcut
```powershell
Ctrl + Shift + Esc
```

---

# Task Manager Views

## Simple View
- Default/basic view
- Shows currently open applications only

## Detailed View
- Click:
  ```text
  More details
  ```
- Displays:
  - Processes
  - Services
  - Startup programs
  - Performance metrics

---

# Important Tabs

| Tab | Purpose |
|---|---|
| Processes | Running apps and background tasks |
| Performance | CPU, RAM, Disk, Network usage |
| App History | Resource usage history |
| Startup | Startup applications |
| Users | Logged-in users |
| Details | Advanced process information |
| Services | Windows services management |

---

# Common Actions

## End a Task
1. Select process/application
2. Click:
   ```text
   End Task
   ```

## Check Resource Usage
- Open:
  ```text
  Performance tab
  ```

## Disable Startup Programs
1. Open:
   ```text
   Startup tab
   ```
2. Right-click application
3. Select:
   ```text
   Disable
   ```

---

# Security Importance

Task Manager is useful for:
- Detecting suspicious processes
- Identifying malware behavior
- Monitoring unusual CPU/RAM usage
- Troubleshooting slow systems
