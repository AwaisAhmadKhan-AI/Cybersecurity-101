# The Windows\System32 Folder

## Windows Folder (`C:\Windows`)
- Main directory containing Windows operating system files
- Usually located at:
  ```powershell
  C:\Windows
  ```
- Can exist on another drive/location

---

# Environment Variables

## Definition
- Variables that store system-related information

## Common Information Stored
- OS paths
- Number of processors
- Temp folder locations
- Windows directory path

## Windows Directory Variable
```powershell
%windir%
```

Example:
```powershell
%windir%\System32
```

---

# System32 Folder

## Path
```powershell
C:\Windows\System32
```

## Purpose
- Contains critical Windows system files
- Essential for proper OS operation

## Common Contents
- System executables (`.exe`)
- DLL files (`.dll`)
- Administrative tools
- Drivers
- Core Windows utilities

---

# Important Notes

## Security Warning
- Deleting/modifying files inside `System32` can make Windows unusable
- Always use caution

## Why It Matters
- Many Windows tools and commands are stored here
- Frequently used in:
  - System administration
  - Troubleshooting
  - Cybersecurity
  - Pentesting

---

# Quick Facts

| Item | Description |
|---|---|
| `%windir%` | Environment variable for Windows folder |
| `System32` | Critical Windows system directory |
| Default Location | `C:\Windows\System32` |
| Contains | Core OS files & tools |
| Risk | Incorrect changes may break Windows |

# Common Commands

## Open System32
```powershell
cd C:\Windows\System32
```

## Open Using Environment Variable
```powershell
cd %windir%\System32
```

## List Files
```powershell
dir
```
